# 🔒 Privacy-First SaaS Component: Interactive CV Anonymizer & Search Engine

A privacy-focused backend application designed to help recruiters process CVs while protecting candidates' sensitive information. The system automatically anonymizes resumes, making them searchable without exposing Personally Identifiable Information (PII).

## ✨ Features

* **PII Detection & Redaction** using **Regex** and **spaCy (NLP)** to identify and anonymize names, email addresses, phone numbers, LinkedIn, and GitHub profiles.
* **Secure Candidate Mapping** with cryptographically generated unique identifiers stored in a protected local JSON registry.
* **Anonymous Resume Search** by skills and keywords (e.g., Python, SQL, Docker) while preserving candidate privacy.
* **Optimized OOP Architecture** using `@property` caching to improve performance, reduce file I/O operations, and ensure data integrity.

## 🛠️ Technologies

* Python
* Object-Oriented Programming (OOP)
* spaCy
* PyPDF2
* Regular Expressions (Regex)
* JSON
